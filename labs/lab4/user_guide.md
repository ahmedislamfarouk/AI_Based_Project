# User Guide: Lab 4 (Minimalist)

## 1. Setup
```bash
pip install langchain langchain-openai langchain-community chromadb faiss-cpu sentence-transformers wikipedia arxiv
```

## 2. Usage
- **App 1**: `python app1_customer_support.py` (Answers customer support queries using RAG).
- **App 2**: `python app2_research_assistant.py` (Generates academic research reports from web/arxiv data).
