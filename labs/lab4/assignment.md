# Projects 7 & 8 Implementation Assignment

## Overview

This assignment requires implementation of two advanced LangChain applications based on the MasterClass repository. Students will develop functional applications that demonstrate comprehensive understanding of advanced LangChain techniques including retrieval-augmented generation, agent-based systems, and multi-source information management.

## Applications to Develop

### Application 1: Enterprise Customer Support System

Develop a LangChain-based customer support system that simulates an enterprise support environment.

**Technical Requirements:**

- Document vectorization with appropriate chunking strategies
- Minimum of three distinct product knowledge bases
- Conversation memory implementation for context retention
- At least two specialized support tools/functions
- Structured response formatting
- Error handling and edge case management

### Application 2: Academic Research Assistant

Develop a LangChain-based research system capable of collecting, synthesizing, and organizing information from multiple sources.

**Technical Requirements:**

- Implementation of at least two research tools/sources
- Information synthesis using appropriate chain types
- Structured report generation with configurable formats
- Source tracking and citation management
- Filtering mechanism for information relevance

## Submission Requirements

For each application, submit:

- Complete, well-documented source code
- User guide with setup instructions and example usage scenarios
